•	¿Qué tipo de error crees que es más común en programas reales?

A mi parecer los errores más comunes suelen ser los errores de ejecución, como ValueError o TypeError, ya que dependen de lo que ingrese el usuario o de situaciones inesperadas durante el uso del programa.

•	¿Por qué es importante manejar excepciones?

Porque permite evitar que el programa se detenga bruscamente ante un error, mostrando mensajes claros al usuario y manteniendo el control del flujo del programa. Además, ayuda a que el sistema sea más seguro y confiable.
